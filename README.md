# Blazor - SB Admin 2
This repository contains a Blazor implementation of the popular [SB Admin 2](https://github.com/StartBootstrap/startbootstrap-sb-admin-2) dashboard template. SB Admin 2 is a free, open-source admin dashboard template built with Bootstrap 4, featuring a clean and modern design.

## Features
- Support for Blazor Server
- Responsive design using Bootstrap 4
- Multiple pre-built pages (Dashboard, Charts, Tables, Forms, etc.)
- Reusable Blazor components
- Easy to customize and extend
- Sidebar navigation
- Top navigation bar
- Footer
- Charts and graphs using Chart.js
- Data tables with sorting and pagination
- Open-source license (MIT)
- Sample data for testing and development

## Prerequisites
- [.NET 10.0 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) or later
- Visual Studio 2022, VS Code, or your preferred .NET IDE

## Getting Started
To get started with the Blazor SB Admin 2 template, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Saffarnejad/BlazorSbAdmin2.git
2. Navigate to the project directory:
   ```bash
   cd BlazorSbAdmin2
3. Restore the NuGet packages:
   ```bash
   dotnet restore
4. Run the application:
   ```bash
   dotnet run
5. Open your web browser and navigate to `https://localhost:5001` to view the dashboard.
6. Customize the template by modifying the Blazor components and styles as needed.
